# CelenganKu

Requirements:
- C++ compiler (tested with MSVC)
- CMake (version 3.10 or higher)

How to compile
```bash
mkdir build
cd build

# for MSVC 
cmake .. -G "NMake Makefiles"
nmake
```

Run Server
```
./CelenganKu_server
```

Run Client
``` bash
# on a different terminal

./CelenganKu_client
```
